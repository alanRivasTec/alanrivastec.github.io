Párrafos HTML

Un párrafo siempre comienza en una nueva línea, y suele ser un bloque de texto.

Párrafo en HTML

   
                El elemento HTML define un párrafo.<p>
                
                Un párrafo siempre comienza en una nueva línea, y los navegadores agregan automáticamente un espacio 
                en blanco (un margen) antes y después de un párrafo.

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>

                                <p>Este es un ejemplo de un parrafo hecho con HTML.</p>
                                <p>Este es un ejemplo de un parrafo hecho con HTML.</p>
                            </body>
                            </html>

Reglas horizontales HTML

                La etiqueta define un salto temático en una página HTML, y es más frecuente se muestra como una 
                regla horizontal.<hr>

                El elemento se utiliza para separar contenido (o definir un cambio) en un HTML página:<hr>

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>
                                <h1>Este es el apartado 1</h1>
                                <p>Este es un simple texto.</p>
                                <hr>

                                <h2>Este es el apartado 2</h2>
                                <p>Este es un simple texto.</p>
                                <hr>

                                <h2>Este es el apartado 3</h2>
                                <p>Este es un simple texto.</p>
                            </body>
                            </html>

Este es el apartado 1

Este es un simple texto.

Este es el apartado 2

Este es un simple texto.

Este es el apartado 3

Este es un simple texto.

 
                La etiqueta es una etiqueta vacía, lo que significa que no tiene una etiqueta final.<hr>

Saltos de línea HTML

                El elemento HTML define un salto de línea.<br>

                Utilícelo si desea un salto de línea (una nueva línea) sin comenzar un nuevo párrafo:<br>

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>

                                <p>Los saltos<br>de linea<br>son muy utiles para muchas cosas.</p>

                            </body>
                            </html>

El elemento HTML <pre>

                El elemento HTML define el texto con formato previo.<pre>

                El texto dentro de un elemento se muestra en una fuente de ancho fijo (normalmente Courier), 
                y conserva tanto los espacios como los saltos de línea:<pre>

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>

                                <p>La etiqueta pre conserva tanto los espacios como los saltos de línea:</p>

                                <pre>

                                    casi todos mis cabellos y poquísimas canas

                                    pero el grave problema es que antes
                                    
                                    no me fijaba en estos detalles.
                                </pre>

                            </body>
                            </html>
