Estilo

            El atributo HTML se utiliza para agregar estilos a un elemento, como color, fuente, tamaño, etc.style

El atributo HTML Style

   
                Establecer el estilo de un elemento HTML, se puede hacer con el atributo.style
                El atributo HTML tiene la siguiente sintaxis:style

Color de fondo

   
                La propiedad CSS define el color de fondo para un elemento HTML.background-color

Ejemplo

Establezca el color de fondo rojo a la etiquete <div>

Código Resultado 
                            <td style="background-color: red;">
                                <h1>Hola buenas tardes</h1>
                            </td>

Hola buenas tardes

Color del texto

   
                La propiedad CSS define el color del texto para un elemento HTML:color

Ejemplo

Código Resultado 
                            <h2 style="color: magenta;">Encabezado color magenta</h2>

Encabezado color magenta

Fuentes

   
                La propiedad CSS define la fuente que se va a utilizar para un elemento HTML:font-family

Ejemplo

Código Resultado 
                            <p style="font-family:courier;">Este es un ejemplo</p>

Este es un ejemplo

Tamaño del texto

   
                La propiedad CSS define el tamaño del texto para un elemento HTML:font-size

Ejemplo

Código Resultado 
                            <h5 style="font-size:160%;">Lenguaje de Programación</h5>
Lenguaje de Programación

Alineación de texto

   
                La propiedad CSS define la alineación horizontal del texto de un elemento HTML:text-align

Ejemplo

Código Resultado 
                            <p style="text-align:center;">Este es un ejemplo de un párrafo centrado</p>

Este es un ejemplo de un párrafo centrado
