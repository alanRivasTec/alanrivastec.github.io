Encabezados HTML

Los encabezados HTML son títulos o subtítulos que desea mostrar en una página web.

Encabezados HTML

   
                Los encabezados HTML se definen con las etiquetas to.<h1><h6>

                <h1> define el encabezado más importante. define el título menos importante.<h6>

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>

                                <h1>Programación web 8AISC. </h1>
                                <h2>Programación web 8AISC. </h2>
                                <h3>Programación web 8AISC. </h3>
                                <h4>Programación web 8AISC. </h4>
                                <h5>Programación web 8AISC. </h5>
                                <h6>Programación web 8AISC. </h6>

                            </body>
                            </html>

Los encabezados son importantes

                Los motores de búsqueda utilizan los encabezados para indexar la estructura 
                y el contenido de sus páginas web.

                Los usuarios a menudo hojean una página por sus encabezados. Es importante utilizar 
                encabezados para mostrar la estructura del documento.

                <h1> Los encabezados deben usarse para los encabezados principales, seguidos de 
                encabezados, luego los menos importantes , y así sucesivamente.<h2><h3>

                Nota: Utilice encabezados HTML sólo para encabezados. No uses encabezados para hacer 
                que el texto sea GRANDE o en negrita.

Encabezados más grandes

   
                Cada encabezado HTML tiene un tamaño predeterminado. Sin embargo, puede especificar el tamaño 
                de cualquier título con el atributo, utilizando la propiedad CSS:stylefont-size

Ejemplo

Código Resultado 
                            <!DOCTYPE html>
                            <html>
                            <body>

                            <h1 style="font-size:40px;">Ing. en Sistemas Computacionales</h1>

                            <p>Puede cambiar el tamaño de un título con el atributo style mediante 
                            la propiedad font-size.</p>

                            </body>
                            </html>
