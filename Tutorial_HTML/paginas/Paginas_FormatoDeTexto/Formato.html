Formato de Texto HTML

Elementos HTML <b> y <strong>

   
                El elemento HTML define texto en negrita, sin ninguna importancia extra.<b>

                El elemento HTML define el texto con gran importancia. El contenido interior 
                se muestra normalmente en negrita.<strong>

Ejemplo

Código Resultado 
                            <p>Este es un párrafo normal</p>

                            <p><b>Este es un párrafo en negritas</b></p>
    
                            <p><strong>Este párrafo es importante!&;/strong></p>

Este es un párrafo normal

Este es un párrafo en negritas

Este párrafo es importante!

Elemento HTML <small>

   
                El elemento HTML define Texto más pequeño:<small></small>

Ejemplo

Código Resultado 
                            <p>Este es un ejemplo</p>
                            <p><small>Este es un ejemplo</small></p>

Este es un ejemplo

Este es un ejemplo

Elemento HTML <mark>

   
                El elemento HTML define el texto que deben marcarse o resaltarse:<mark></mark>

Ejemplo

Código Resultado 
                            <p>Me gustan los <mark>caramelos</mark></p>

Me gustan los caramelos

Elemento HTML <del>

   
                El elemento HTML define el texto que se ha eliminado de un documento. Los navegadores 
                generalmente marcarán una línea a través del texto eliminado:<del>

Ejemplo

Código Resultado 
                            <p>Me divierte jugar contigo <del>al futbol</del></p>

Me divierte jugar contigo al futbol

Elemento HTML <ins>

   
                El elemento HTML define un texto que se ha insertado en un documento. Los navegadores 
                generalmente subrayarán insertados Mensaje de texto:<ins></ins>

Ejemplo

Código Resultado 
                            <p>Soy un estudiante  <del>de univesidad </delGt; <ins>de la 
                            carrera de ISC</ins>. </p>

Soy un estudiante de univesidad de la carrera de ISC.

Elemento HTML <sub>

   
                El elemento HTML define Texto de subíndice. El texto del subíndice aparece medio carácter por 
                debajo de la línea normal, y a veces se representa en una fuente más pequeña. El texto de 
                subíndice se puede utilizar para fórmulas químicas.

Ejemplo

Código Resultado 
                            <p>H<sub>2</sub>O</p>

H2O

Elemento HTML <sup>

   
                El elemento HTML define Texto en superíndice. El texto en superíndice aparece medio carácter 
                por encima de lo normal , y a veces se representa en una fuente más pequeña. Se puede utilizar 
                texto en superíndice para notas al pie.

Ejemplo

Código Resultado 
                            <p>Fórmula: 2<sup>3</sup></p>

Fórmula: 23
