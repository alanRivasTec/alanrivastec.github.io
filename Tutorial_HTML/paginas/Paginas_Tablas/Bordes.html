Bordes

            Las tablas HTML pueden tener bordes de diferentes estilos y formas.

Como agregar bordes a una tabla 

   
                Cuando agrega un borde a una tabla, también agrega bordes alrededor de cada celda de tabla:

                Para agregar un borde, use la propiedad CSS en , y los elementos:bordertablethtd

Ejemplo

Código Resultado 
                            <table style=" border: 1px solid black;width:100%">

                                </tr>
