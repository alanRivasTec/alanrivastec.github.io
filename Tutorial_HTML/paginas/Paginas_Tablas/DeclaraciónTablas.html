Tablas HTML

            Las tablas HTML permiten a los desarrolladores web organizar los datos en filas y Columnas.

Definir una tabla HTML 

   
                Una tabla en HTML consta de celdas de tabla dentro de filas y columnas.

                Celdas de tabla
                    Cada celda de la tabla se define mediante una etiqueta y una etiqueta.<td></td>

                Filas de tabla
                    Cada fila de tabla comienza con una y termina con una etiqueta.<tr></tr>

                Encabezados de tabla
                    A veces desea que las celdas sean celdas de encabezado de tabla. En esos casos, use la 
                    etiqueta en lugar de la etiqueta:<th><td>

Ejemplo

Código Resultado 
                            <table style="width:100%">
                                <tr>
                                    <th>País</th>
                                    <th>Continente</th>
                                    <th>Capital</th>
                                </tr>
                                <tr>
                                    <td>México</td>
                                    <td>America</td>
                                    <td>CDMX</td>
                                </tr>
                                <tr>
                                    <td>Alemania</td>

                                </tr>
                                <tr>
                                    <td>Japón</td>

                            </table>
