Listas ordenadas HTML

            La etiqueta HTML define una lista ordenada. Un La lista ordenada puede ser numérica o 
            alfabética.<ol>

Lista HTML ordenada 

   
                Una lista ordenada comienza con la etiqueta. Cada elemento de lista comienza 
                con la etiqueta.<ol><li>

                Los elementos de la lista se marcarán con números de forma predeterminada:

Ejemplo

Código Resultado 
                            <p> Un ejemplo de una lista ordenada </p> 

                            <ol> 
                            <li> Programación web </li> 
                                <li> LA2 </li>
                                <li> PSP </li> 
                            </ol>

Un ejemplo de una lista ordenada

  1. Programación web
  2. LA2
  3. PSP

Lista HTML ordenada: el atributo Type 

   
                El atributo de la etiqueta, define el tipo de la Marcador de elemento de lista:type<ol>

Ejemplo

Código Resultado 
                            
    
                                <p>Lista ordenada con letras </p> 

                                <ol type="A"> 
                                    <li>Tacos </li> 
                                    <li>Enchiladas </li> 
                                    <li>Guisado </li> 
                                </ol>

Lista ordenada con letras

  1. Tacos
  2. Enchiladas
  3. Guisado

Listas HTML anidadas 

   
                    Las listas se pueden anidar (lista dentro de la lista):

Ejemplo

Código Resultado 
                                <ol>
                                    <li>Color</li>
                                    <li>Tamaño
                                        <ol>
                                            <li>Chico</li>
                                            <li>Grande</li>
                                        </ol>
                                        </li>
                                    <li>Forma</li>
                                </ol>
  1. Color
  2. Tamaño
    1. Chico
    2. Grande
  3. Forma
