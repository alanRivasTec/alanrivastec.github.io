Listas HTML desordenadas

            La etiqueta HTML define un desordenado Lista (con viñetas).<ul>

Lista HTML desordenada 

   
                Una lista desordenada comienza con la etiqueta <ul>. Cada elemento de lista 
                comienza con la etiqueta.<li>

                Los elementos de la lista se marcarán con viñetas (pequeños círculos negros) de forma predeterminada:

                La propiedad CSS se utiliza para definir el estilo de la Marcador de elemento de lista. 
                Puede tener uno de los siguientes valores:list-style-type

Ejemplo

Código Resultado 
                            <ul style="list-style-type: circle;">
                                <li>Motorola</li>
                                <li>Nokia</li>
                                <li>Xiaomi</li>
                            </ul>
  • Motorola
  • Nokia
  • Xiaomi

Lista HTML anidadas 

   
                Las listas se pueden anidar (lista dentro de la lista):

Ejemplo

Código Resultado 
                            <ul>
                                <li>Color</li>
                                <li>Tamaño
                                    <ul>
                                        <li>Chico</li>
                                        <li>Grande</li>
                                    </ul>
                                </li>
                                <li>Forma>/li>
                            </ul>
  • Color
  • Tamaño
    • Chico
    • Grande
  • Forma
