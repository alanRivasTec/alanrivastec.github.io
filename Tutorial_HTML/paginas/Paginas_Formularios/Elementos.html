Elementos de Formularios HTML


El elemento <input>

   
                Uno de los elementos de formulario más utilizados es el elemento. <input>

                El elemento se puede mostrar de varias maneras, dependiendo del atributo.<input>type

Ejemplo

Código Resultado 
                            >form action="prueba.html">
                                >label for="fname">Edad:>/label>>br>
                                >input type="number" id="fname" name="fname">>br>>br>
                                >input type="submit" value="Enviar">
                            >/form>



El elemento <select>

                El elemento define una lista desplegable:<select>

Ejemplo

Código Resultado 
                            <form action="prueba.html">
                                <label for="carros">Elige una marca de carro:</label>
                                <select id="carros" name="carros">
                                <option value="Ford">Ford</option>
                                <option value="GMC">GMC</option>
                                <option value="Honda">Honda</option>
                                <option value="Kia">Kia</option>
                                </select>
                                <input type="submit">
                            </form>

Código Resultado

El elemento <datalist>

                El elemento especifica una lista de opciones predefinidas para un elemento.<datalist><input>

                Los usuarios verán una lista desplegable de las opciones predefinidas a medida que 
                ingresan datos.
                    
                El atributo del elemento, debe referirse al atributo del elemento.list<input>id<datalist>

Ejemplo

Código Resultado 
                            <form action="prueba.html">
                                <input list="Lenguajes">
                                <datalist id="Lenguajes">
                                    <option value="Java">
                                    <option value="HTML">
                                    <option value="C">
                                    <option value="Python">
                                    <option value="CSS">
                                </datalist>
                            </form>
