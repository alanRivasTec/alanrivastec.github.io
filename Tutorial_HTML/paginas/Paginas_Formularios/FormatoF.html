Formularios HTML

            Se utiliza un formulario HTML para recopilar la entrada del usuario. La entrada del usuario 
            es La mayoría de las veces se envía a un servidor para su procesamiento.

El elemento <forma>

   
                El elemento HTML se utiliza para crear un formulario HTML para la entrada 
                del usuario:<form>

                El elemento es un contenedor para diferentes tipos de elementos de entrada, tales como: 
                campos de texto, casillas de verificación, radio botones, botones de envío, etc.<form>

El elemento <input>

                El elemento HTML es el más Elemento de formulario utilizado. <input>
                
                Un elemento se puede mostrar en de muchas maneras, dependiendo del atributo.<input>type

                Tipo	                                     Descripción
                <input type="text">	Muestra un campo de entrada de texto de una sola línea
                <input type="radio">	Muestra un botón de opción (para seleccionar una de las muchas opciones)
                <input type="checkbox">	Muestra una casilla de verificación (para seleccionar cero o más de muchas opciones)
                <input type="submit">	Muestra un botón de envío (para enviar el formulario)
                <input type="button">	Muestra un botón en el que se puede hacer clic

Campos de texto

                El define un campo de entrada de una sola línea para Entrada de texto.<input type="text">

Ejemplo

Código Resultado 
                            <form>
                                <label for="fname">Nombre:</label><br>
                                <input type="text" id="fname" name="fname" value="Luis"><br>
                                <label for="lname">Edad:</label><br>
                                <input type="text" id="lname" name="lname" value="23">
                            </form>



El elemento <label> 

                La etiqueta define una etiqueta para muchos elementos de formulario.<label>

                El elemento es útil para usuarios de lectores de pantalla, porque el lector de pantalla 
                leerá en voz alta la etiqueta cuando El usuario se centra en el elemento de entrada.<label>

                El define un botón de opción.<input type="radio">

                Los botones de opción permiten al usuario seleccionar UNA de un número limitado de opciones.

El botón Enviar 

                El define un botón para enviar los datos del formulario a un controlador de 
                formulario.<input type="submit">

                El controlador de formulario suele ser un archivo en el servidor con un script para procesar 
                datos de entrada.

                El controlador de formulario se especifica en el atributo del formulario.action

Ejemplo

Código Resultado 
                            <form action="prueba.html">
                                <label for="fname">Nombre: </label> <br>
                                <input type="text" id="fname" name="fname" value="Enrique"> <br>
                                <label for="lname">Edad: </label> <br>
                                <input type="number" id="lname" name="lname" value="12"> <br> <br>
                                <input type="submit" value="Enviar">
                            </form>
