Atributos de Formularios HTML

El atributo Action

   
                El atributo define la acción que se realizará cuando se envíe el formulario.action

                Por lo general, los datos del formulario se envían a un archivo en el servidor cuando el 
                usuario hace clic en el botón Enviar.

Ejemplo

Código Resultado 
                            <form action="prueba.html">
                                <label for="fname">Nombre:</label><br>
                                <input type="text" id="fname" name="fname" value="Alan"><br>
                                <label for="lname">Edad:</label><br>
                                <input type="number" id="lname" name="lname" value="12"><br><br>
                                <input type="submit" value="Enviar">
                            </form>





El atributo target

                El atributo especifica dónde Mostrar la respuesta que se recibe después de enviar el 
                formulario.target

                El atributo puede tener uno de los Valores siguientes:target

                El valor predeterminado es lo que significa que La respuesta se abrirá en la ventana actual._self

Ejemplo

Código Resultado 
                            <p>Al enviar este formulario, el resultado se abrirá en una nueva pestaña 
                            del navegador: </p>

                            <form action="prueba.html" target="_blank">
                                <label for="fname">Nombre: </label> <br>
                                <input type="text" id="fname" name="fname" value="Juan"> <br>
                                <label for="lname">Edad: </label> <br>
                                <input type="number" id="lname" name="lname" value="34"> <br> <br>
                                <input type="submit" value="Enviar">
                            </form>

Al enviar este formulario, el resultado se abrirá en una nueva pestaña del navegador:

El atributo Method

                El atributo especifica el HTTP Método que se utilizará al enviar los datos del formulario.method

                Notas sobre GET:

                    * Anexa los datos del formulario a la dirección URL, en pares nombre/valor.
                    * ¡NUNCA use GET para enviar datos confidenciales! (¡los datos del formulario enviado son 
                    visibles en la URL!).
                    * La longitud de una URL es limitada (2048 caracteres).
                    * Útil para envíos de formularios en los que un usuario desea marcar el resultado.
                    * GET es bueno para datos no seguros, como cadenas de consulta en Google.
                
                Notas sobre POST:

                    * Anexa los datos del formulario dentro del cuerpo de la solicitud HTTP (el los datos del 
                    formulario no se muestran en la dirección URL).
                    * POST no tiene limitaciones de tamaño y se puede utilizar para enviar grandes 
                    cantidades de datos.
                    * Los envíos de formularios con POST no se pueden marcar como favoritos.

El elemento <label> 

                La etiqueta define una etiqueta para muchos elementos de formulario.<label>

                El elemento es útil para usuarios de lectores de pantalla, porque el lector de pantalla 
                leerá en voz alta la etiqueta cuando El usuario se centra en el elemento de entrada.<label>

El atributo Novalidate 

                El atributo es un atributo booleano.novalidate

                Cuando está presente, especifica que los datos del formulario (entrada) no deben 
                validarse cuando se envían.

                <form action="/action_page.php" novalidate></form>
