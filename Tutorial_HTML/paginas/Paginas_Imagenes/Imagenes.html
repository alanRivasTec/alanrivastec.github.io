Imágenes HTML

            Las imágenes pueden mejorar el diseño y la apariencia de una página web.

Sintaxis de imágenes HTML

   
                La etiqueta HTML se utiliza para incrustar un imagen en una página web.<img>

                Las imágenes no se insertan técnicamente en una página web; Las imágenes están vinculadas 
                a la web .pages. La etiqueta crea una retención espacio para la imagen a la 
                que se hace referencia.<img>

                La etiqueta está vacía, solo contiene atributos y no tener una etiqueta de cierre.<img>

                La etiqueta tiene dos obligatorios Atributos:<img>

                src: especifica la ruta de acceso a la imagen
                alt: especifica un texto alternativo para la imagen

                <img src="url" alt="alternatetext">

Ejemplo

Código Resultado 
                            <img src="flor.webp" alt="Es una flor">

Tamaño de la imagen: ancho y alto

   
                Puede utilizar el atributo para especificar el ancho y altura de una imagen.style

Ejemplo

Código Resultado 
                            <img src="flor.webp" alt="Es una flor" style="width:300px;height:200px;"≫

Como alternativa, puede utilizar los atributos y:width height

Código Resultado
<img src="flor.webp" alt="flor" width="300" height="250">
