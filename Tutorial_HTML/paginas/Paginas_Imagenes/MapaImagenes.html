Mapas de imágenes HTML

            Con los mapas de imágenes HTML, puede crear áreas en las que se puede hacer clic en una imagen.

Mapas de imágenes

   
                La etiqueta HTML define un mapa de imagen. Un mapa de imagen es una imagen con áreas en las 
                que se puede hacer clic. Las áreas se definen con una o más etiquetas. <map><area>

Ejemplo

Intente hacer click en la taza o el télefono

Código Resultado 
                            >img src="workplace.jpg" alt="Workplace" usemap="#workmap">

                            >map name="workmap">
                            >area shape="rect" coords="34,44,270,350" alt="Computadora" href="computer.htm">
                            >area shape="rect" coords="290,172,333,250" alt="Celular" href="phone.htm">
                            >area shape="circle" coords="337,300,44" alt="Cafe" href="coffee.htm">
                            >/map>
