                Para mostrar un vídeo en HTML, utilice el elemento:<video></video>

                Cómo funciona
                
                El atributo agrega controles de vídeo, como reproducir, pausar y volumen.controls

                Es una buena idea incluir siempre y atributos. Si no se establecen el alto y el ancho, 
                la página puede parpadear mientras se carga el video.widthheight

                El elemento le permite especificar video alternativo archivos entre los que el navegador puede 
                elegir. El navegador utilizará el primer formato reconocido.<source>

                El texto entre las etiquetas y solo se mostrará en navegadores que no lo hacen Apoyar el 
                elemento.<video></video><video>
