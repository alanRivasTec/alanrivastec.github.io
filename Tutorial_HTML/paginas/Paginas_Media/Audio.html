El elemento HTML <audio> 

   
                Para reproducir un archivo de audio en HTML, utilice el elemento:<audio>

                Cómo funciona

                El atributo agrega controles de audio, como reproducir, pausar y volumen.controls

                El elemento le permite especificar audio alternativo archivos entre los que el navegador 
                puede elegir. El navegador utilizará el primer formato reconocido.<source>

                El texto entre las etiquetas y solo se mostrará en navegadores que no lo hacen 
                Apoyar el elemento.<audio></audio><audio>
