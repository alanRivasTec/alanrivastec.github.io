Videos HTML de YouTube

Reproducir un vídeo de YouTube en HTML 

   
                Para reproducir el vídeo en una página web, haga lo siguiente:

                Sube el vídeo a YouTube
                Toma nota del video id
                Define un elemento en tu página web<iframe>
                Dejar que el atributo apunte a la URL del vídeosrc
                Utilice los atributos y para especificar la dimensión del reproductorwidth height
                Agregue cualquier otro parámetro a la URL (ver más abajo)

Ejemplo

Código Resultado 
                            <iframe width="420" height="345" src="https://www.youtube.com/embed/tgbNymZ7vqY">
                            </iframe>
